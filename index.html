Step 1: Include the Script

Add this single line to your HTML file, preferably before the closing </body> tag:

<script src="https://cdn.chatbot-widget.io/v1/chatbot.min.js"></script>

Step 2: Initialize the Widget

Create a script tag to configure and initialize the chatbot:

<script> window.ChatbotConfig = { apiKey: 'YOUR_API_KEY_HERE', position: 'bottom-right', theme: 'light' }; </script>

Step 3: Get Your API Key

  1. Sign up at https://chatbot-widget.io/dashboard
  2. Create a new project
  3. Copy your API key from the dashboard
  4. Replace YOUR_API_KEY_HERE with your actual key

ðŸ’¡ Tips & Troubleshooting

Common Issues

  • Widget not appearing: Check that your API key is correct and active
  • CORS errors: Ensure your domain is whitelisted in the dashboard
  • Styling conflicts: The widget uses scoped styles to avoid conflicts
  • Performance issues: Use the minified version in production

Best Practices

  • Load the script asynchronously for better performance
  • Use the minified version (chatbot.min.js) in production
  • Test on mobile devices to ensure responsive design
  • Monitor API usage from your dashboard
  • Keep your API key secure and never expose it client-side in sensitive applications
